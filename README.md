### 데이터 모델링
현실 세계에 존재하는 데이터를 컴퓨터 세계의 데이터 베이스로 옮기는 과정

### 데이터 모델링 순서
1. 요구사항 분석
    1. 엔티티(객체) 찾기
        1. 객체 - 명사
        2. 관계 - 동사
    2. 속성 찾기
    3. 관계 찾기
    4. 요구사항 정의서 작성
2. 개념적 모델링
    1. ERD 그리기
3. 논리적 모델링
    1. Excel Table 정의서 작성
    2. PK, FK 설정
    3. RM(Relation Model) 만들기
4. 물리적 모델링
    1. 테이블 정의서
    2. SQL Script 작성
    3. 최종 PM(Physical Model) 만들기
5. DBMS 구현
    1. Reverse Engineering
    2. Forward Engineering
